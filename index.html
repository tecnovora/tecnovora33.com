    ¿Qué es arquitectura hexagonal en programación?

    La arquitectura hexagonal, a la que también se le conoce como arquitectura de puertos y adaptadores, es una arquitectura de software que se basa en la idea de aislar la lógica comercial central de las preocupaciones externas, por medio de la separación de la aplicación en componentes débilmente acoplados. Pero, ¿por qué no lo ilustramos a través de un ejemplo? Imaginemos que estamos creando una API que necesita obtener algunos datos de una API REST.
